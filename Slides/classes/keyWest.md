# What about notes?

Major scale: [0,1,2,3,4,5,6]

Common patterns:
* Scales: every step
* Octave: notes repeat every 7 steps
* Chords: every other note [0,2,4]

Melodies?
* Geometric shapes
	* triangles
	* sawtooths
	* stair steps
* Call and response
	* density and space
	* theme and variation
	* tension & resolution



---

# What about harmony?

We can play different chords just by using an offset, e.g.:
* [0,2,4] [1,3,5] [2,4,6] [3,5,7]

We can use standard harmony notation
* Theory.progression = 'I I IV V'

We can change keys
* Theory.key = 'D'
* for minor keys use lowercase: Theory.key = 'd'
