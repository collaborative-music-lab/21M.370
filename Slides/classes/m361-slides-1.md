## Course Modules

1. **Materiality of Sound** — waveform vs spectrum; timbre; space; recording/editing basics.
2. **Grids & Loops** — tempo, meter, quantization, phase, polymeter/polyrhythm.
3. **Algorithmic Music** — rule-based systems, mappings, constraints; determinism vs randomness.
4. **TBD** Spatial distribution of sound, Generative composition, Composing for robots, Noise, . . .

## Digital Audio Workstation

- Professional audio recording, mixing, and mastering
- Any DAW will do
- Pro Tools First, Reaper, Logic Audio, etc.

## Reaper

- Relatively new DAW (2005)
- Incorporates some new tools and techniques into the DAW world, like scripting
- Inexpensive, but no free option
- 60-day free trial, potentially repeatable

## Live Coding

- Browser-based Environment: JavaScript-based, evaluates code to schedule/process audio.  
- Core concepts: patterns, transforms, mapping, transport.
- ianhattwick.com/creativitas

## Live Coding (2): Workflow
<!-- .slide: data-visibility="hidden" -->

- Evaluate selected code blocks; keep changes atomic.  
- Deterministic variations > uncontrolled randomness.  
- Save snippets; comment parameters; measure-level iteration.

## Headphones

- Closed-back recommended for tracking/editing.  
- Safe listening: avoid long sessions above ~85 dB SPL.  
- Adapters/cables labeled; check polarity and fit.
